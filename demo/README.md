# Demo Index

本目录用于展示 HydroAgent-OS 在典型水文水利任务中的真实对话流程、参数推断逻辑与最终工程结论。

## 如何使用本目录

1. 按下表选择你的关注场景（产汇流 / 水工 / 频率分析 / 调洪 / 率定思路）。
2. 打开对应 `demo_x.md` 查看完整对话链路。
3. 到 `../output/demo_x/` 查看配套图件和结果文件。

## 推荐阅读路径（给首次访问者）

- 路径 A（快速理解能力）：`demo_1` → `demo_6` → `demo_2`
- 路径 B（关注交互流程）：`demo_7` → `demo_5`
- 路径 C（关注工程决策）：`demo_4` → `demo_2` → `demo_6`

## 输出解释口径（阅读时可参考）

- 模型过程：先看输入是否完整，再看参数是否经确认后执行。
- 结果质量：重点关注峰值、峰现时间、削峰率、NSE/KGE/RMSE 等指标。
- 工程意义：看结论是否给出风险边界与可执行建议。
